# 🏅 Olympics Data Analysis & Medal Prediction App

## Overview

An interactive Streamlit-based web application for exploring Olympic history, analyzing medal trends, visualizing athlete performance, and predicting future medal outcomes using Machine Learning.

## Features

### 📊 Olympics Analysis

* Medal Tally Dashboard
* Overall Olympic Statistics
* Country-wise Analysis
* Participation Trends
* Men vs Women Participation Analysis

### 🏃 Athlete Analytics

* Athlete Profile Search
* Career Statistics
* Medal History
* Event Participation Analysis

### 🧬 Machine Learning

* Athlete Medal Prediction
* Country Medal Prediction
* Athlete Clustering using K-Means
* Random Forest, Logistic Regression, and XGBoost Models

### 📈 Data Visualization

* Interactive Plotly Charts
* Heatmaps
* Scatter Plots
* Trend Analysis
* Country Performance Visualizations

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
